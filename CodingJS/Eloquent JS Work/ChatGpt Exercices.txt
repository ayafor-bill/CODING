Chapter 1: Values, Types, and Operators

    String Reversal
        Write a function reverseString that takes a string as an argument and returns the string reversed without using built-in methods like reverse().

    Array Manipulation
        Create a function removeDuplicates that takes an array of numbers and returns a new array with duplicates removed, preserving the original order.

    Object Comparison
        Write a function deepEqual that takes two values (which can be objects or arrays) and returns true if they are deeply equal and false otherwise. Consider nested structures.

Chapter 2: Program Structure

    Grading System
        Write a function grade that takes a score (0-100) and returns a letter grade based on the following scale:
            A: 90-100
            B: 80-89
            C: 70-79
            D: 60-69
            F: 0-59

    FizzBuzz Variation
        Extend the FizzBuzz function to allow custom divisors. Create a function customFizzBuzz that takes an array of divisor/string pairs (e.g., [[3, "Fizz"], [5, "Buzz"]]) and a range, returning the appropriate FizzBuzz results for that range.

    Finding Prime Numbers
        Write a function findPrimes that takes a number and returns an array of all prime numbers up to that number.

Chapter 3: Functions

    Higher-Order Function
        Create a function map that takes an array and a callback function as arguments. It should return a new array where each element is the result of applying the callback to each element of the original array.

    Function Currying
        Write a function curriedAdd that takes one argument and returns another function that takes another argument, allowing for partial application (e.g., curriedAdd(2)(3) should return 5).

    Memoization
        Implement a memoization function memoize that takes a function as an argument and returns a new function that caches the results of the original function to optimize repeated calls with the same arguments.

Additional Challenges

    Flattening Arrays
        Write a function flatten that takes a nested array (an array that can contain other arrays) and returns a single, flat array.

    Creating a Basic Calculator
        Implement a function calculate that takes a string input like "2 + 3 * (4 - 1)" and evaluates the expression, returning the result. You may use the eval function or write a simple parser.

    Tally Votes
        Write a function tallyVotes that takes an array of candidate names and returns an object with each candidate's name as a key and their vote count as the value.
