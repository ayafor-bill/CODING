# CodingPython Glossary Script

This folder contains Python practice files and simple examples. One of the scripts, `glossary.py`, is an interactive glossary helper that explains common Python terms.

## `glossary.py`

`glossary.py` is a beginner-friendly program that:

- stores definitions for basic Python terms in a dictionary
- prompts the user to choose a term
- displays the selected definition
- allows the user to exit by typing `quit` or `exit`

### Supported terms

- `Get`
- `Modules`
- `Def`
- `Loop`
- `List`

### How to run

```bash
python glossary.py
```

Then type one of the supported terms when prompted.

### Notes

- The script accepts input case-insensitively.
- If a term is not recognized, it asks the user to try again.
- This is a simple example of using dictionaries, user input, and control flow in Python.
