# Python By Example Challenges

This folder contains beginner Python exercises that appear to come from the book *Python By Example* by Nicholas H. Tollervey.

## Book source

The code in `Challenge1.py` is organized like introductory practice problems from the book **Python By Example**. The folder name and the style of simple input/output tasks strongly suggest that this is based on exercises from that beginner-friendly title.

## About `Challenge1.py`

`Challenge1.py` is a small interactive script that walks the user through a set of simple tasks:

1. Prompt the user for first name and surname.
2. Display a greeting using the entered names.
3. Print a joke.
4. Ask for two numbers and compute their sum.
5. Ask for a third number and compute a multiplication involving the first two.
6. Ask how many pizza slices the user started with and how many they have eaten, then calculate the remaining slices.
7. Ask the user for their current age and display what their age will be on their next birthday.

## How to run

Run the script with Python from the `PythonbyExample` folder:

```bash
python Challenge1.py
```

Then follow the on-screen prompts.

## Notes

- This script uses `input()` to collect data from the user.
- It converts numeric answers to integers using `int()` before performing arithmetic.
- The exercise is typical of early chapters in a beginner Python book, focusing on console interaction and basic operations.
